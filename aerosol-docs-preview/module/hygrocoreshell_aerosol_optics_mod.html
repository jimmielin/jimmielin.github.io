type, public, extends(aerosol_optics) ::  hygrocoreshell_aerosol_optics

hygrocoreshell_aerosol_optics Table look up implementation of aerosol_optics to parameterize aerosol radiative properties in terms of core mass fraction, black carbon/dust fraction, kappa and relative humidity
